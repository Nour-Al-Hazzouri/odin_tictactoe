# Instructions
[TOP's instructions for this project](INSTRUCTIONS.md)
# Goal
Have a working browser TicTacToe game for 2 players with basic functionality
# Required Minimal Functionalities
- Start game
- Receive user inputs (names)
- Display markers where players click
- End game over 3 successions or draw
- Show result after game end
- Restart game
# Desired Code
Have the code to be inside objects, function factories, IIFEs, and reduce global variables and functions as much as possible.
# Components
For this project, the following components are required:
- Player object
    - player1 (name)
    - player2 (name)
- Gameboard object
    - gameboard (array)
    - markers (X, O)
- Game object
    - start (function)
    - end game (function)
    - show results (function)
    - restart (function)

Variables and functions must be inside these objects depending on the logic as much as possible.
