# Odin TicTacToe

**The good ol' TicTacToe game!**

This project is the second project of TOP's Fullstack JS path.

## Project Goal
Aside from the nostalgia, this project's goal was to experience the concepts of closures, factory functions, and IIFEs. These concepts emerged after prototypes due to the shortcomings of JS's prototypes, which raised security risks despite their memory-efficient nature.

## What Did I Build This Project With?
My attempt was to use as few global variables as possible. By having each closure with its own related variables, and specific values only changeable through specific methods for that purpose, the chances for incorrect values and reassignment decrease.

## How Did I Build This Project?
I initially checked TOP's [INSTRUCTIONS](INSTRUCTIONS.md), then created another file to brainstorm the logic (literally named it [Brainstorm](Brainstorm.md) XD). As instructed, the first version was done through the terminal, and the rest is just rewiring the logic with DOM objects.
